# 🍽️ ChefAI

A modern full-stack Food Recipe Application that allows users to discover, search, and manage delicious recipes. Users can browse recipes by category, view detailed cooking instructions, save their favorite recipes, and explore a wide variety of cuisines through an intuitive and responsive interface.

---

## 📌 Features

* 🔍 Search recipes by name
* 🥗 Browse recipes by category
* 📖 View detailed recipe information
* ❤️ Save favorite recipes
* 👨‍🍳 Step-by-step cooking instructions
* 🕒 Display preparation and cooking time
* 🥘 Ingredient list with measurements
* 📱 Responsive design for desktop and mobile devices
